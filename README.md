# Gong

A simple meditation timer PWA with customizable duration and sound notifications.

## Features

- Timer presets (5, 10, 15, 20, 30 min) or custom duration
- Optional pre-countdown before meditation starts
- Sound notifications (Bell, Tibetan Singing Bowl, Gong)
- Offline support
- Installable on mobile via "Add to Home Screen"

## Live Demo

https://lawrencedrums.github.io/gong/

## Run Locally

```bash
# Clone the repo
git clone https://github.com/lawrencedrums/gong.git
cd gong

# Serve with any static server
python3 -m http.server 8000
```

Open http://localhost:8000 in your browser.
